A modern black-colored, two-columns, responsive theme optimized for Chinese content. Features mobile-first design, threaded comments, and clean typography.

== Description ==

Bones7456 is a modern, responsive WordPress theme designed specifically for Chinese content. It features a sleek black design with orange accents, making it perfect for blogs and personal websites.

= Key Features =

* **Responsive Design**: Fully responsive layout that works perfectly on desktop, tablet, and mobile devices
* **Mobile-First Approach**: Optimized for mobile viewing with collapsible sidebar
* **Chinese Typography**: Custom font support for better Chinese character display
* **Threaded Comments**: Built-in support for nested comments
* **Clean Code**: Semantic HTML5 and clean CSS
* **Widget Ready**: Multiple widget areas for customization
* **Translation Ready**: Prepared for internationalization
* **SEO Friendly**: Optimized for search engines

= Theme Structure =

The theme includes all standard WordPress template files:
* index.php - Main template
* header.php - Header template
* footer.php - Footer template
* sidebar.php - Sidebar template
* single.php - Single post template
* page.php - Page template
* archive.php - Archive template
* search.php - Search results template
* 404.php - 404 error template
* comments.php - Comments template

= Customization =

The theme supports WordPress Customizer for easy customization of colors, fonts, and layout options.

== Installation ==

1. Upload the theme files to the `/wp-content/themes/bones7456` directory, or install the theme through the 'Appearance > Themes' screen in WordPress.
2. Activate the theme through the 'Appearance > Themes' screen in WordPress.
3. Customize the theme through the 'Appearance > Customize' screen.
